# 🎮 坦克大战 (Tank Battle)

一个用纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏。

## 🎯 游戏特性

- **经典玩法**：控制坦克消灭所有敌人，逐关推进
- **多种墙体**：砖墙可破坏，钢墙不可破坏
- **AI敌人**：具有一定智能的敌方坦克，会追踪玩家和射击
- **道具系统**：速度提升、快速射击、护盾、额外生命
- **关卡系统**：难度递增，地图随机生成
- **粒子效果**：爆炸特效和屏幕震动
- **音效系统**：基于 Web Audio API 的射击和爆炸音效
- **鼠标瞄准**：支持键盘+鼠标双重操作

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| W/A/S/D 或 方向键 | 移动坦克 |
| 空格 / 鼠标左键 | 射击 |
| P | 暂停/继续 |
| R | 重新开始 |

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接在浏览器中打开 `index.html` 即可游玩！

或者直接访问 GitHub Pages：`https://YOUR_USERNAME.github.io/tank-battle/`

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3
- JavaScript (ES6+)
- Web Audio API
